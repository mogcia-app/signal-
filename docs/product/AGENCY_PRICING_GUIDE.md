# Signal. 代行業者向け料金体系ガイド

## 料金体系の記載方法

### 推奨表記（シンプル版）

```
【システム利用料】

■ 代行店アカウント（固定）
月額 6万円（税込）
- 代行店様ご自身のアカウント利用料（固定費）
- システム基本機能すべて利用可能
- 専任サポート含む

■ クライアントアカウント（変動）
1アカウントあたり 月額 2万円（税込）
- 管理するクライアントアカウント数に応じて追加
- クライアントごとのブランド設定・データ管理が可能
- 月次レポート自動生成機能含む

【例：10アカウント運用の場合】
基本料金: 月額 6万円
+ クライアントアカウント: 月額 2万円 × 10 = 20万円
合計: 月額 26万円（税込）

【サポート】
特に、未経験者・SNS代行周りのサービスを販売しているプロに向け
- 専任スタッフによるサポート
- 導入時のレクチャー
- 定期的な改善提案
- メール・電話サポート
```

```
■ 基本料金（代行店アカウント）
月額 6万円（税込）
- システム利用権（固定）
- 基本機能すべて利用可能

■ クライアントアカウント追加料金
1アカウントあたり 月額 2万円（税込）
- 管理するクライアントアカウント数に応じて変動

【総額の計算例】
10アカウント運用の場合:
基本料金 6万円 + (2万円 × 10アカウント) = 月額 26万円（税込）
```

### 代替案2（段階的料金型）

```
【システム利用料】

■ 代行店アカウント（固定）
月額 6万円（税込）
- システム基本利用料（固定）

■ クライアントアカウント（変動）
- 1〜5アカウント: 1アカウントあたり 月額 2.5万円（税込）
- 6〜10アカウント: 1アカウントあたり 月額 2万円（税込）
- 11アカウント以上: 1アカウントあたり 月額 1.5万円（税込）

【例：10アカウント運用の場合】
基本料金: 月額 6万円
+ クライアントアカウント: 月額 2万円 × 10 = 20万円
合計: 月額 26万円（税込）
```

## 記載時のポイント

### 1. 明確な区別
- **固定費**と**変動費**を明確に分ける
- 「代行店アカウント（固定）」と「クライアントアカウント（変動）」という言葉で区別

### 2. 具体例を提示
- 実際の運用例（例：10アカウント運用）での総額を明示
- 計算方法をわかりやすく示す

### 3. 価値の明確化
- 基本料金に含まれる機能・サポートを明記
- クライアントアカウント追加時に何ができるかを明記

### 4. サポートの強調
- 未経験者・SNS代行サービスを販売しているプロ向けであることを明記
- サポート内容を具体的に記載

## よくある質問への回答例

### Q1. クライアントアカウント数はいつでも変更できますか？
A. はい、月単位で変更可能です。追加・削除は月末までにご連絡いただければ、翌月から反映されます。

### Q2. 基本料金だけで何アカウントまで運用できますか？
A. 基本料金（6万円）は代行店様ご自身のアカウント利用料です。クライアントアカウントは1アカウントあたり追加料金がかかります。

### Q3. 初期費用はかかりますか？
A. 初期費用は別途かかる場合がございます。詳細はお問い合わせください。

### Q4. 解約はできますか？
A. はい、いつでも解約可能です。解約は月末までにご連絡いただければ、翌月末日で解約となります。
