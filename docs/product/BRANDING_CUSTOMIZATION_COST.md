# Signal. ブランディング・デザイン変更コスト見積もり

## 📋 前提条件

- **開発費は既に支払い済み**（自社開発済み）
- **機能・ロジックはそのまま**（コードベースは変更なし）
- **変更するのはブランディング・デザインのみ**

---

## 🎨 変更が必要な項目

### 1. ブランディング要素
- ロゴ（SVG、PNG、各種サイズ）
- ファビコン（favicon-32.png など）
- アプリ名・サービス名（"Signal" → 新名称）
- メタデータ（title、description、OGP画像）

### 2. デザイン要素
- カラースキーム（CSS変数、Tailwind設定）
- フォント（Geist → 新フォント or そのまま）
- UI要素の色（ボタン、カード、グラフなど）
- ヒートマップの色スケール

### 3. テキスト・コンテンツ
- 画面内の"Signal"表記（約20ファイル）
- エラーメッセージ、通知メッセージ
- ドキュメント内の表記

### 4. 設定・デプロイ
- Firebase設定（projectId、authDomain など）
- 環境変数設定
- デプロイ・テスト

## 📋 内訳詳細

### 1. ブランディング変更（50-80時間）
- **ロゴ作成・配置**: 20-30時間
  - ロゴデザイン（既存ロゴを提供する場合は10-15時間）
  - 各種サイズの生成（favicon、OGP画像など）
  - ファイル配置
- **メタデータ変更**: 10-15時間
  - `layout.tsx`のmetadata更新
  - OGP画像の設定
  - 各種メタタグの確認
- **ファビコン設定**: 10-15時間
  - 各種サイズのファビコン生成
  - `layout.tsx`への設定
- **確認・調整**: 10-20時間

### 2. デザイン調整（100-150時間）
- **カラースキーム変更**: 40-60時間
  - `globals.css`のCSS変数変更
  - Tailwind設定の変更
  - 105個のコンポーネントの確認・調整
- **フォント変更（オプション）**: 20-30時間
  - フォントの選択・設定
  - `layout.tsx`のフォント設定変更
  - 表示確認
- **UI要素の色調整**: 30-40時間
  - ボタン、カード、モーダルの色
  - グラフ、チャートの色（Chart.js設定）
  - ヒートマップの色スケール
- **確認・調整**: 10-20時間

### 3. テキスト置換（30-50時間）
- **一括置換**: 10-15時間
  - "Signal" → 新名称の一括置換
  - コード内のコメント、変数名は除外
  - ユーザー向けテキストのみ変更
- **個別確認・調整**: 15-25時間
  - 約20ファイルの確認
  - 文脈に応じた調整
  - 置換漏れの確認
- **確認・テスト**: 5-10時間

### 4. 設定・デプロイ（40-60時間）
- **Firebase設定**: 15-20時間
  - 新プロジェクトの作成（または既存プロジェクトの設定変更）
  - `firebase.ts`の設定変更
  - 環境変数の設定
- **環境変数設定**: 10-15時間
  - `.env.local`の設定
  - 各種APIキーの設定
- **デプロイ・動作確認**: 15-25時間
  - ビルド・デプロイ
  - 動作確認
  - エラー対応

### 5. テスト・確認（30-50時間）
- **全画面の動作確認**: 20-30時間
  - 各ページの表示確認
  - 機能の動作確認
  - レスポンシブ対応確認
- **デザインの一貫性確認**: 10-20時間
  - カラースキームの一貫性
  - ロゴの表示確認
  - UI要素の確認

### 6. ドキュメント更新（20-30時間）
- **README更新**: 10-15時間
  - プロジェクト名の変更
  - 説明文の更新
- **設定ファイルの更新**: 10-15時間
  - `package.json`のname
  - その他の設定ファイル

## 🎯 推奨価格設定

### **400万円 - 600万円**

**内訳:**
- **カスタマイズ費用**: 300万円 - 450万円
- **ライセンス費用**: 100万円 - 150万円（コードベースの使用権）

**理由:**
1. **開発コスト（4,500万円 - 6,000万円）の約10%**
   - 既に開発済みのコードベースの使用権として適正価格
2. **カスタマイズ工数（270-420時間）の対価**
   - ブランディング変更の実作業コスト
3. **市場価値**
   - 同様の機能を1から開発する場合の約10%のコスト
   - 短期間（1-2ヶ月）で提供可能

## ⚠️ 注意点

1. **知的財産権**
   - コードベースの使用権を明確に定義

2. **サポート範囲**

3. **再配布・転売の制限**

   - 既存データの移行が必要な場合の費用
   - データの所有権を明確に
