# Signal プロジェクト アピールポイント

## 📋 プロジェクト概要

**Signal** は、Instagram運用を「分析 → 学習 → 改善 → 生成」まで一気通貫で支援する、AI主導のグロースハックツールです。  
運用者が入力する投稿データやフィードバック、AIが算出する指標を **MasterContext** として蓄積し、次の一手に直結するプランやコンテンツ生成までを自動化します。

---

## 🎯 主要機能・特徴

### 1. **AI主導の月次レポート**
- 自動解析による「今月のまとめ」「先月比較」「注目ポイント」「次のアクションプラン」をワンカードで提示
- PDCA実行度・振り返りデータ件数・改善反映率などPDCAメトリクスを可視化
- リスク・異常検知と投稿タイプ別インサイトを自動検出

### 2. **学習ダッシュボード（MasterContext）**
- フィードバック・アクションログを一覧化し、継続的に学習
- 学習バッジ、タイムライン、ポストパターン要約、成功/改善ギャラリーを表示
- Deep diveカードでクラスタ情報や派生指標、AIサマリーを確認可能
- 過去の成功パターンを記憶し、次の提案に活用

### 3. **投稿単位の深掘り分析**
- 各投稿の詳細パフォーマンス分析（クラスタ比較、差分指標、AIサマリー）
- 保存済みAIサマリーと最新生成の比較機能
- アクションログ・フィードバック・コメントログを一画面で管理

### 4. **AI投稿生成・構成支援**
- 投稿エディタ + AIのテンプレート支援、構成生成、画像ヒント提案
- コメント返信アシスト（投稿コンテキストを渡し、トーン別の返信案を生成）
- リール・ストーリー固有の構成生成API連携

### 5. **統合分析フォーム**
- 投稿分析フォーム＋統計カード＋AIインサイトの3カラム構成
- コメント＆返信ログ、満足度フィードバック、AIによる深掘り要約を連携
- 分析データのリセットボタンで重複データや誤入力を即クリア

### 6. **KPI分解とドリルダウン**
- リーチ、保存、フォロワー、エンゲージメント、総合インタラクション数などの詳細分析
- 時間帯別ヒートマップ、ハッシュタグ分析、投稿タイプ別パフォーマンス
- 貢献度が高い投稿の自動抽出と詳細リンク

### 7. **計画・シミュレーション機能**
- 投稿頻度/予算/目標変更などのシミュレーション
- AI戦略生成APIと連携し、MasterContextを活用した提案
- リアルな代替目標のワンボタン生成、異常検知からのシナリオ再提案

## 🚀 差別化ポイント

### 1. **7種類以上のデータを統合分析**
一般的なツールが「フォロワー数・エンゲージメント率のみ」を参照するのに対し、Signalは以下を統合分析：
- 過去30件の詳細投稿パフォーマンス
- 業種特性データ
- 成長トレンド
- フィードバック・アクションログ
- コメント・返信履歴
- 時間帯・曜日別パフォーマンス
- ハッシュタグ効果分析

### 2. **具体的で実行可能な提案**
- 抽象的なアドバイスではなく、**具体的な投稿タイトル・キャプション・ハッシュタグ**まで提示
- **投稿タイミングの具体的な提案**（例：水曜日14時-16時が最適）
- **数値予測付きの改善案**（例：月間エンゲージメント率2.5%→3.5%への向上が期待）

### 3. **継続的な学習と進化**
- **MasterContext**によるユーザー固有の成功パターン記憶
- **PDCAサイクル自動化**で継続的に改善
- データが蓄積されるほど提案が最適化される

### 4. **コストパフォーマンス**
- 一般的なAIツール: 数百円/回
- **Signal: 1社あたり月10円程度**（毎日60回使用しても月10円）
- 無制限の投稿生成とデータ分析が標準装備

### 5. **代替案の自動提案**
目標が困難と判定された場合、自動的に3つの代替案を提示：
- 現実的な目標（推奨）
- 挑戦的な目標
- 段階的アプローチ

### 6. **データ修正性**
- 分析データのリセット機能で重複や誤入力を即クリア
- データ整合性を維持する補正スクリプト
- PDCAの検証がクリアに保たれる
