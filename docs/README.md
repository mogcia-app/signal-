# Docs Index

このディレクトリには運用・開発ノートをカテゴリ別にまとめています。  
各フォルダには Markdown ファイルがまとまっているので、目的に応じて参照してください。

## ディレクトリ一覧

- `sentry/` — Sentry のセットアップ、運用ノウハウ、障害対応ログ
- `ai/` — AI 分析/生成に関する仕様やトラブルシューティング
- `ops/` — 本番運用・移行計画・環境構築・テスト関連ドキュメント
- `dev/` — 開発フロー、Lint/Git ルール、コードレビューガイド
- `product/` — LP/HP の改善案、差別化資料、デモ準備ノート

必要に応じてサブディレクトリを追加し、関連ドキュメントをまとめてください。***
