# /onboarding 仕様変更について（Admin向け）

作成日: 2026-01-30

## 概要

`/onboarding`ページのUIと役割を明確化するため、以下の変更を実施しました。

---

## 変更内容

### 1. セクションタイトルの変更

**変更前:**
- 「目標と課題」

**変更後:**
- 「基本方針・長期目標」

**理由:**
- 月ごとに変わる目標は運用計画ページ（`/instagram/plan`）で設定できるようになったため、`/onboarding`の目標は「基本方針・長期目標」として位置づけを明確化

### 2. 説明文の追加

**追加内容:**
```
この目標は、あなたのビジネスの根幹となる基本方針です。
月ごとに変わる目標は、運用計画ページで設定できます。
```

**表示位置:**
- 「基本方針・長期目標」セクションのタイトル直下

**目的:**
- ユーザーに`/onboarding`と`/instagram/plan`の役割の違いを明確に伝える

### 3. Instagram AI設定の表示変更

**変更前:**
- 以下の項目が表示されていた：
  - トーン
  - マナー・ルール
  - 注意事項・NGワード
  - 運用目標
  - 活動の動機
  - 機能
  - その他参考情報

**変更後:**
- 「注意事項・NGワード」のみを表示
- 表示位置：「基本方針・長期目標」セクションの下

**理由:**
- 「注意事項・NGワード」はAI生成時に必須遵守事項として最優先で参照される重要な情報
- その他の項目（トーン、マナー、運用目標など）は、AI生成時に参照されるが、ユーザーが直接確認する必要は低い

## データ構造への影響

### 保存されるデータ（変更なし）

以下のデータは引き続き保存されます：

**`userProfile.businessInfo`:**
- `goals`: 目標（配列）
- `challenges`: 課題（配列）
- その他の既存フィールド（業種、会社規模、ターゲット市場など）

**`userProfile.snsAISettings.instagram`:**
- `cautions`: 注意事項・NGワード（表示される）
- `manner`: マナー・ルール（非表示だが保存・参照される）
- `tone`: トーン（非表示だが保存・参照される）
- `goals`: 運用目標（非表示だが保存・参照される）
- `motivation`: 活動の動機（非表示だが保存・参照される）
- `features`: 機能（非表示だが保存・参照される）
- `additionalInfo`: その他参考情報（非表示だが保存・参照される）

**重要:**
- データ構造自体は変更されていません
- 既存のデータはそのまま有効です
- UIでの表示のみが変更されています

## ユーザーへの影響

### 既存ユーザー

- **影響なし**: 既存のデータはそのまま有効
- **表示の変化**: `/onboarding`ページを開くと、表示内容が変更されていることを確認できます

### 新規ユーザー

- **基本方針の明確化**: 「基本方針・長期目標」として、長期にわたって変わらない目標を設定することを理解しやすくなります
- **運用計画との関係性**: 月ごとに変わる目標は運用計画ページで設定できることを理解しやすくなります

---

## 運用計画ページ（/instagram/plan）との関係

### 役割の明確化

**`/onboarding`（基本方針・長期目標）:**
- ビジネスの根幹となる基本方針
- 変更頻度: 低い（数ヶ月〜数年単位）
- 例: 「ブランド認知度の向上」「顧客との関係構築」

**`/instagram/plan`（今月の目標）:**
- 月ごとに変わる具体的な目標
- 変更頻度: 高い（月ごとに変更可能）
- 例: 「今月は新商品の認知度向上に重点を置く」

### データの優先順位

AI生成時の優先順位：
1. **最優先**: `/instagram/plan`で設定した`monthlyGoals`と`monthlyChallenges`
2. **フォールバック**: `/onboarding`の`goals`と`challenges`

## Admin側での確認事項

### 1. 既存データの確認

以下のコレクションのデータが正常に保存されているか確認：
- `users`コレクションの`businessInfo.goals`と`businessInfo.challenges`
- `users`コレクションの`snsAISettings.instagram.cautions`

### 2. ユーザーサポート時の注意点

**よくある質問への対応:**

**Q: 「目標と課題」が「基本方針・長期目標」に変わったけど、データは消えていない？**
A: データは消えていません。表示名が変更されただけです。既存の目標と課題はそのまま有効です。

**Q: Instagram AI設定の項目が表示されなくなったけど、設定は消えた？**
A: 設定は消えていません。UIでの表示のみを変更しました。AI生成時には引き続き参照されます。

**Q: 月ごとに目標を変えたい場合はどうすればいい？**
A: 運用計画ページ（`/instagram/plan`）で「今月の目標設定」セクションから設定できます。

### 3. データ移行の必要性

**不要です。** データ構造の変更はなく、既存データはそのまま有効です。
