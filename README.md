# Signal.

Next.js + Firebase Functions を使ったモダンなWebアプリケーション

## 概要

Signal.は、Instagram運用を最適化するためのAI搭載プラットフォームです。

## 技術スタック

- **フレームワーク**: Next.js
- **バックエンド**: Firebase Functions
- **データベース**: Firestore
- **認証**: Firebase Authentication
- **AI**: OpenAI API

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
