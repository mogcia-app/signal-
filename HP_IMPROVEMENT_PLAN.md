# Signal. HP改善プラン

## 📊 現状分析

### 主な問題点

#### 1. **情報過多・焦点の散漫**

- セクションが多すぎて、何が重要か分からない
- 「3つの仕組み」「6つの核心価値」「こんな企業様にオススメ」など、情報が分散
- ユーザーが迷子になりやすい

#### 2. **CTA（行動喚起）の配置・メッセージ**

- 「お問い合わせ」「無料診断」など複数のCTAが散在
- 優先順位が不明確
- メッセージが統一されていない

#### 3. **コンバージョン要素の不足**

- 具体的な実績・数値が少ない（「90%自動化」以外に具体的数字がない）
- お客様の声・事例が不足
- 信頼性を示す要素（ロゴ、認定など）が不足

#### 4. **コンテンツの重複**

- 「従来vs Signal.」の比較が複数箇所に分散
- 同じ情報が異なる表現で繰り返されている
- 冗長で読み進めるモチベーションが下がる

#### 5. **デザイン・構造**

- セクション間の関連性が見えにくい
- スクロールが長すぎる
- 視覚的な階層が不明確

## 🎯 改善方針

### 優先順位1: 構造の簡素化と焦点の明確化

#### 新しい構造案

```
1. ヒーローセクション（ファーストビュー）
   → 強力なキャッチコピー + 主要なCTA 1つ

2. 3つの核心価値（最も重要な差別化要素）
   → 「90%自動化」を具体的に

3. お客様の課題と解決（Before/After）
   → 6つのお悩みを3つに集約

4. 実績・信頼性（新規追加）
   → 数字、ロゴ、お客様の声

5. 料金プラン
   → シンプルに3プラン

6. 導入フロー
   → 5ステップを簡略化

7. ファイナルCTA
   → 「無料相談」のみ
```

### 優先順位2: コンバージョン最適化

#### 追加すべき要素

- **実績・数値**: 導入企業数、達成率、ROIなど
- **お客様の声**: 短いコメント + 企業名・役職
- **ビフォーアフター**: 具体的な数値で改善事例
- **デモ動画/スクリーンショット**: 実際の画面を見せる
- **信頼性**: パートナー企業、認定、メディア掲載など

### 優先順位3: コンテンツの最適化

#### 改善ポイント

1. **専門用語を減らす**
   - 「KPI逆算型PDCA」→「目標達成までの自動設計」
   - 「カルテ」→「成長する提案システム」

2. **ベネフィット重視**
   - 機能説明より、「何ができるか」を強調
   - 「時間削減」「売上向上」「コスト削減」など具体的な成果

3. **ストーリーテリング**
   - お客様の「困りごと→解決→成果」の流れ
   - 感情に訴える構成

## 📝 具体的な改善案

### セクション1: ヒーロー（ファーストビュー）

**現状の問題**

- キャッチコピーが弱い
- CTAが目立たない

**改善案**

```
【キャッチコピー】
「SNS運用、まだ手作業で消耗してますか？」
「90%を自動化。残り10%で売上を創る。」

【サブコピー】
Instagram、X、TikTok、YouTubeの投稿から分析まで
AIが一括管理。あなたは戦略に集中するだけ。

【CTA】
大きなボタン: 「無料で始める（30日間無料）」
小さなテキスト: 「デモを見る（動画リンク）」
```

### セクション2: 3つの核心価値（「6つ」から「3つ」に集約）

**現状の問題**

- 6つは多すぎる
- 差別化ポイントが弱い

**改善案**

```
1. AIが90%自動化
   → 投稿提案・作成・分析まで自動化
   → 具体的な数値（例：月20時間削減）

2. 目標から逆算設計
   → KPI設定から最適な戦略を自動提案
   → 従来との比較図

3. 成長する提案システム
   → 使えば使うほど精度が上がるAI
   → データ蓄積の仕組みを図解
```

### セクション3: お客様の課題と解決

**現状の問題**

- 6つの課題は多すぎる
- 解決方法が抽象的

**改善案**

```
【3つの課題に集約】

1. 「投稿しても伸びない」
   → Before: いいね10件/投稿
   → After: いいね500件/投稿（実績）

2. 「売上に繋がらない」
   → Before: フォロワー1万、問い合わせ月3件
   → After: フォロワー1.5万、問い合わせ月30件（実績）

3. 「運用に手が回らない」
   → Before: 週3時間の運用作業
   → After: 週30分で運用可能（90%削減）

【各課題に実績・数値を追加】
```

### セクション4: 実績・信頼性（新規追加）

**現状の問題**

- 実績・信頼性のセクションが不足

**改善案**

```
【実績】
- 導入企業数: XX社
- 平均達成率: XX%
- 平均ROI: XX倍

【お客様の声】
- 3-5件の短いコメント
- 企業名・役職・業種

【信頼性】
- パートナー企業ロゴ
- メディア掲載
- 認定・受賞歴（あれば）
```

### セクション5: 料金プラン

**現状の問題**

- 3プランが並列で分かりにくい
- 「おすすめ」の理由が不明確

**改善案**

```
【推奨プランを見やすく】
- 2SNSプラン（おすすめ）を大きく表示
- 「なぜおすすめか」を明記
- 30日間無料を強調

【比較表で分かりやすく】
- 機能比較表
- 「すべてのプランに含まれる」を明記
```

### セクション6: 導入フロー

**現状の問題**

- 5ステップは多すぎる
- 時間感覚が不明確

**改善案**

```
【4ステップに簡略化】
1. お問い合わせ（1分）
2. 無料診断（30分）
3. カスタマイズ（2-3週間）
4. 運用開始（サポート付き）

【各ステップに所要時間を明記】
```

### セクション7: ファイナルCTA

**現状の問題**

- CTAが多すぎる

**改善案**

```
【1つの明確なCTA】
「無料で始める（30日間無料トライアル）」
+ 電話番号（緊急対応用）

【心理的ハードルを下げる】
- 「クレジットカード不要」
- 「すぐに始められる」
- 「専門スタッフがサポート」
```

## 🎨 デザイン改善案

### 1. スクロール長の削減

- 現在: 約3000px（長すぎる）
- 目標: 2000px以内（約30%削減）

### 2. 視覚的な階層

- フォントサイズの差を大きく
- 余白を適切に配置
- カラーコントラストを強化

### 3. モバイル最適化

- セクションごとの表示を最適化
- CTAボタンを大きく
- 読みやすさを重視

## 📈 コンバージョン最適化

### A/Bテスト案

1. **CTAのメッセージ**
   - 「無料相談を始める」
   - 「30日間無料で試す」
   - 「デモを見る」

2. **料金表示**
   - 月額を大きく表示
   - 年額も併記（割引表示）

3. **実績表示**
   - 数値のみ
   - 数値 + 説明
